## Suggested study order

1. Read **01** for the big picture, then skim **GLOSSARY** for unfamiliar words.  
2. Use **02** and **03** when you need attack vocabulary.  
3. Use **04** before interviews or compliance discussions.  
4. Keep **reference/TIPS_AND_EXAMPLES.md** open for spaced repetition.
