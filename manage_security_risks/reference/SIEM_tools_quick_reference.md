# SIEM tools — quick reference

**Full detail (definition + every SIEM named + what each does and how it’s used):**  
[../notes/10_siem_tools_definition_and_products.md](../notes/10_siem_tools_definition_and_products.md)

| SIEM product (in your notes) | One line |
|------------------------------|----------|
| **Splunk® Enterprise** | Self-hosted: retain, analyze, search log data; security insight and alerts in real time. |
| **Splunk® Cloud** | Cloud-hosted: collect, search, monitor log data (same product family). |
| **Chronicle** (Google SecOps) | Cloud-native Google SIEM: retain, search, detect; IOC and ingestion dashboards. |

**Not SIEM:** **Suricata** (network detection → feeds SIEM); **SOAR** (automation next to SIEM).
