## Suggested study order

1. Skim **[GLOSSARY](glossary/GLOSSARY.md)** for fast recall, then read **01** for full context on data classification and assets.  
2. Read **02** to connect classification and handling to **who** gets escalated issues (data roles and compliance).  
3. Read **03** to map **security stakeholders** and how day-to-day reporting reaches senior leaders.  
4. Read **04** for **how** to communicate: be direct, follow protocol, tailor to role, pick the right channel.  
5. Read **05** for **visual** communication and **dashboards** when data tells the story (pairs with **04**).  
6. Read **06** to prepare for **technical interviews** (Python, concepts, sample questions).
