# Linux distributions for security analysts

Previously, different **Linux distributions** were introduced—including **KALI LINUX** ™. (KALI LINUX ™ is a trademark of OffSec.) This note covers additional distributions security analysts should know.

---

## KALI LINUX ™

**KALI LINUX** ™ is an **open-source** distribution widely used in the security industry. It is **Debian-based** and **pre-installed** with many tools for **penetration testing** and **digital forensics**.

- **Penetration test** — A simulated attack that helps identify vulnerabilities in systems, networks, websites, applications, and processes.  
- **Digital forensics** — Collecting and analyzing data to determine what happened after an attack.

These activities are central to security work. **KALI LINUX** ™ is not the only Linux distribution used in cybersecurity.

---

## Ubuntu

**Ubuntu** is an open-source, **user-friendly** distribution used widely in security and other industries. It offers both a **command-line interface (CLI)** and a **graphical user interface (GUI)**. Ubuntu is **Debian-derived** and includes common applications by default; users can install more—including security tools—via a **package manager**.

Ubuntu has a **large community** of resources to support users. It is also widely used for **cloud computing**; as organizations move to cloud servers, security work may more often involve **Ubuntu derivatives**.

---

## Parrot

**Parrot** is an open-source distribution commonly used for security. Like **KALI LINUX** ™, it ships with **pre-installed** tools for penetration testing and digital forensics. Like **KALI LINUX** ™ and Ubuntu, it is **Debian-based**.

Parrot is considered **user-friendly** because of its **GUI**, which many find easy to navigate, in addition to its **CLI**.

---

## Red Hat® Enterprise Linux®

**Red Hat Enterprise Linux** is a **subscription-based** distribution built for **enterprise** use. Unlike the distributions above, **Red Hat** is **not free**. Because it is built and supported for enterprise use, Red Hat provides a **dedicated support team** customers can contact for issues.

---

## AlmaLinux

**AlmaLinux** is a **community-driven** Linux distribution created as a **stable replacement** for **CentOS**.

**CentOS** was an open-source distribution closely related to **Red Hat**. Its final stable release, **CentOS 8**, was in **December 2021**. CentOS used source code published by Red Hat to offer a similar platform.

**AlmaLinux** is intended as a **drop-in replacement** for CentOS 8 so that applications and configurations that worked on CentOS can **continue to work** on AlmaLinux.

---

## Key takeaways

**KALI LINUX** ™, **Ubuntu**, **Parrot**, **Red Hat**, and **CentOS** are all widely used Linux distributions. Security analysts should recognize these and others they may see in their careers.
